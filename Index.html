Complete Coding Learning Guide

Chapter 1: HTML

HTML ka full form: HyperText Markup Language. Ye website ka structure banata hai.

Basic Tags:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a href="URL"> - Links
  • <img src="image.jpg"> - Images
  • <ul> / <ol> - Lists
  • <table> - Tables
  • <button> - Buttons

Example:

<!DOCTYPE html>
<html>
<head>
  <title>My Website</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is my first website.</p>
  <a href="https://google.com">Visit Google</a>
</body>
</html>

Chapter 2: CSS

CSS ka full form: Cascading Style Sheets. Ye website ko design aur style deta hai.

Important Properties:

  • color - Text color
  • background-color - Background color
  • font-size - Font size
  • margin - Element ke bahar space
  • padding - Element ke andar space
  • border - Border design
  • display:flex - Flexible layout

Example:

body{
  background-color: #e0f7ff;
  color: #002a7a;
  font-family: Arial, sans-serif;
}

h1{
  text-align: center;
  color: #004aad;
}

Chapter 3: JavaScript

JavaScript website ko interactive aur dynamic banata hai.

Important Concepts:

  • Variables: let, const
  • Functions: function myFunc(){}
  • Loops: for, while
  • If-else conditions
  • DOM Manipulation: document.getElementById()

Example:

<button onclick="hello()">Click Me</button>

<script>
function hello(){
  alert("Hello World");
}
</script>

Chapter 4: Python

Python ek high-level language hai. Ye AI, automation aur data science me use hoti hai.

Important Concepts:

  • Variables: x = 10, name = "Aryan"
  • If-Else: Conditional statements
  • Loops: for, while
  • Functions: def myFunc():
  • Lists: [1,2,3], Dictionaries: {"name":"Aryan"}

Example:

print("Hello World")

name = "Aryan"
age = 15

if age > 10:
    print("Student")

for i in range(5):
    print(i)

def add(a, b):
    return a + b

Chapter 5: C Language

C ek system programming language hai.

Important Concepts:

  • Variables: int, float, char
  • Loops: for, while
  • If-Else conditions
  • Functions

Example:

#include <stdio.h>

int main(){
  printf("Hello World");
  return 0;
}

Chapter 6: C++

C++ object-oriented programming language hai.

Important Concepts:

  • Class & Objects
  • Variables & Functions
  • Loops & Conditions
  • Inheritance, Polymorphism (Advanced)

Example:

#include <iostream>
using namespace std;

class Student{
public:
  string name;
  int age;
};

int main(){
  Student s1;
  s1.name = "Aryan";
  s1.age = 15;

  cout << "Hello " << s1.name << endl;
  return 0;
}
